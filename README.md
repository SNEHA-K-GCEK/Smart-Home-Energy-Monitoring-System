# Smart Home Energy Monitoring System

## Overview

This project monitors electrical energy consumption in real time and estimates electricity cost.

## Features

- Real-time energy monitoring
- Power calculation
- Energy calculation
- Cost estimation
- High consumption alerts
- CSV logging
- PDF reports
- Dashboard visualization

## Technologies

- Python
- Streamlit
- Pandas
- ReportLab
- ESP32 (optional)
